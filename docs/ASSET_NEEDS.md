# Asset Needs - Prototype Phase

Untuk tahap movement dan leveling world, asset bisa placeholder dulu. Kalau kamu mau mulai ganti visual, ini urutan asset yang paling berguna.

## Character

- `Nara` sprite sheet: idle, walk, jump/fall. Side view, ukuran konsisten.
- Opsional awal: Bimo idle/talk untuk marker NPC pertama.

## Background 2D

- Jalan utama Kota Ranting sore, format wide side-scrolling/layered.
- Layer parallax: sky, distant building/hill, street front.
- Set toko kecil: toko listrik, toko kue, toko bunga, klinik, taman festival.

## Props

- Lampu jalan, kabel, papan toko, pot bunga, bangku, poster kesehatan, kardus makanan, kotak P3K.

## UI

- Dialog box frame.
- Objective panel kecil.
- Button prompt `E`.

## Audio Later

- Footstep.
- Ambient sore/malam.
- Bell toko.
- Lamp light on.

## Format Yang Enak Buat Godot

- PNG transparan untuk karakter dan props.
- Background PNG per layer.
- Sprite sheet dengan ukuran frame jelas, misalnya 64x96 atau 128x128 per frame.
- Nama file rapi, contoh: `nara_walk_64x96.png`, `street_evening_bg.png`.
